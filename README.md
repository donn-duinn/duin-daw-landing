# DUIN DAW — AI-Powered Music Studio

Built by my kid tonight. An AI music production app — like Ableton, but you can talk to it.

**What it does:**
- Multi-track timeline + session view (like Ableton Live)
- AI assistant — say "make a trap beat at 140 BPM" and it builds it
- 34 music scales, 40+ chord types, 25 progressions
- 15 genre templates (techno, house, DnB, hip-hop, trap, lofi...)
- Built-in synths, effects, piano roll, MIDI
- Runs on Linux, Mac, Windows

**Price:** $29 one-time. No subscriptions.

**Stack:** Electron, Tone.js, FastAPI, Python
